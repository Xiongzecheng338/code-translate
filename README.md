## 📌 项目核心定位
CIPHER NEXUS是一款集**加密实验、解密分析、操作归档、知识体系**于一体的轻量化密码学全栈工具平台，聚焦古典与现代加密技术的工程化实现与技术解析。平台原生支持**16种主流密码类型**，基于前端原生技术实现本地化运算，搭载智能密码算法识别引擎、批量密文处理模块、本地存储归档系统，实现明文-密文的双向转化、密文的智能破解与操作轨迹的全生命周期管理，为密码学爱好者、技术研究者、安全从业者提供高可用、无依赖的密码学实验环境，解锁从**应用层操作**到**原理层解析**的全维度密码学探索能力。

## 🚀 核心技术模块解析
平台采用**模块化架构**设计，各模块解耦且协同，基于原生技术实现无插件、无服务端依赖的本地运算，保障数据安全性与操作高效性，核心技术模块如下：

### 🔒 加密实验室 | 明文→密文 全维度加密实现
作为平台核心运算模块，实现**古典加密、现代简易加密、哈希计算**的一体化工程化落地，核心技术特性：
- 支持**双向加密**：基于对称加密逻辑实现明文到密文的转化，同时支持加密参数的自定义配置，适配不同加密强度需求；
- 集成**哈希计算引擎**：实现主流哈希算法的原生实现，输出固定长度的数字指纹，满足数据完整性校验需求；
- 无限文本容量解析：突破文本长度限制，支持大文本量的高效加密运算，运算效率随本地硬件性能线性提升；
- 无依赖本地运算：所有加密逻辑在前端本地执行，数据不落地服务端，保障明文数据的安全性。

### 🔓 解密分析 | 密文→明文 智能破解体系
搭载**智能密码类型识别引擎**，实现密文与加密算法的自动匹配，为密文破解提供技术支撑，核心技术特性：
- 智能算法识别：基于正则表达式、字符特征分析、算法特征匹配等技术，自动识别密文对应的加密算法类型，减少人工判断成本；
- 批量密文处理：支持多组密文的批量导入与解析，实现批量解密，提升密文处理效率；
- 解密过程可视化：输出密文破解的核心步骤与逻辑，从技术层解析解密原理，而非单纯的结果输出；
- 多类型密文兼容：覆盖古典密码（凯撒、维吉尼亚、栅栏等）、现代简易加密的全类型密文破解。

### 📜 历史档案 | 操作轨迹 本地化全生命周期管理
基于**HTML5 LocalStorage**技术实现操作记录的本地归档，打造无服务端的操作轨迹管理体系，核心技术特性：
- 本地无感知存储：所有加密、解密操作记录自动归档至本地，无需手动保存，数据仅储存在本地设备，无泄露风险；
- 记录快速恢复：支持历史操作记录的一键检索、恢复与重新运算，实现操作轨迹的回溯；
- 全量记录管理：支持单条/批量记录的删除，实现操作记录的精细化管理，释放本地存储资源。

### 📚 密码图鉴 | 密码学知识 体系化技术解析
构建**古典→现代**密码学的全维度知识体系，是平台的技术支撑模块，核心技术特性：
- 算法原理深度解析：从数学逻辑、工程实现、核心参数三个维度，拆解每一种加密算法的底层原理；
- 技术知识库沉淀：涵盖密码学基础概念、算法分类、适用场景、安全特性等核心知识，形成可查阅的密码学知识库；
- 与实验模块联动：知识解析与实验操作一一对应，实现**学→练**一体化的密码学学习模式。

## ⚡ 系统核心技术特性
1. **16+密码类型全覆盖**：兼容古典加密、现代简易加密、哈希算法等多类型密码学技术，满足多样化实验需求；
2. **本地原生运算**：基于前端原生技术实现所有加密/解密逻辑，无插件、无服务端依赖，数据本地闭环；
3. **智能算法识别引擎**：基于字符特征与算法逻辑的智能匹配，实现密文算法的自动识别，提升解密效率；
4. **批量并行处理**：支持多组密文/明文的批量操作，实现并行运算，适配批量处理场景；
5. **无限文本容量**：无明文/密文的长度限制，支持大文本量的高效运算；
6. **操作记录本地归档**：基于LocalStorage实现操作记录的全量存储，支持快速检索与恢复；
7. **响应式多终端适配**：兼容电脑、平板、手机等多终端，实现跨设备的密码学实验。

## 🛠️ 快速技术上手指南
平台采用**无门槛工程化设计**，无需配置开发环境、无需安装插件，仅需浏览器即可实现全功能操作，步骤如下：
1. **访问实验平台**：打开浏览器，访问[CIPHER NEXUS 密码学实验室](https://badhope.github.io/code-translate/)，进入系统主界面，无需注册/登录，直接使用；
2. **加密操作**：进入「加密实验室」，选择加密算法/哈希计算类型，输入明文，按需配置加密参数，点击运算按钮，一键生成密文，结果支持一键复制；
3. **解密分析**：进入「解密分析」模块，输入密文/批量导入密文，系统自动识别加密算法，点击解密按钮，输出明文结果与解密过程解析；
4. **历史回溯**：进入「历史档案」，查看所有加密/解密操作记录，支持单条记录的恢复、复制与删除；
5. **技术学习**：进入「密码图鉴」，查阅密码学算法原理与知识体系，夯实实验的技术基础。
